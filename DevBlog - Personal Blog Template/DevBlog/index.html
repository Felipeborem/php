7 de abril de 2021 por Felipe Borem


Como eu sou uma pessoa muito legal vou explicar o que cada uma dessas extenções/pluguins servem

  • 1° auto rename tag: Ela server para completar uma tag html
  • 2° Beautfy: Deixa o codigo bunitin
  • 3° Boostrap 4: ele serve para recunhecer o boostrap para completar no html
  • 4° Community material Theme: Te fornece ulguns temas para o editor ficar mais bunitin
  • 5° Debugger for Chome: Nunca uso, e nem sei pra que serve
  • 6° IntelliSense for css names in HTML: Lenbra da extenssão do boostrap, a mesma coisa mas só com o css
  • 7° Live share: faz com que 2 pessoas editem em tempo real o codigo
  • 8° Matarial Icon Theme: cria os simbolos de acordo com o formato do arquivo (no lado esquerdo)
  • 9° Matarial Theme: tem mais uns temas para o editor de texto
